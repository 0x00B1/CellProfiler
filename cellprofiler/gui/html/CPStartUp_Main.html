Getting Started

Welcome to CellProfiler! CellProfiler is automated image analysis software designed to measure biological phenotypes in images. If you're new to CellProfiler, then this guide is for you.

Unfamilar with the interface? See our interface summary for a quick overview. Afterwards, you can refer to the following resources to help get you started.

Resources

Read the documentationOur manual is available here. Also, detailed help is available for any module by clicking the button, or using the Help menu in the toolbar.
Try example pipelinesOur Examples page on our website has pipelines and images of various biological assays. Pick the ones that most resemble your phenotypes of interest and begin adjusting the pipeline settings.
Watch our tutorialsVideos of CellProfiler demos are available on the Tutorials page.
Go online for user supportIf you still need help, you can post your question on our online forum.

Once you are accustomed to using CellProfiler and you no longer need this starting page, you can turn it off under File > Preferences...
