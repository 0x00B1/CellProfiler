Summary of the Interface

When you start CellProfiler, it displays the main interface, which has graphical tools for managing images, pipelines and modules. The interface is divided into four main parts, as shown in the following illustration:

Interface tool

Description
Pipeline panelLists the modules in the pipeline, as well as module testing and display options. Below are buttons for adjusting module positions and getting help.
File panelLists images and pipeline files in the current input folders.
Settings panelContains the available options for each module.
Folder panelSets and/or creates the default input and output folders and output filename.
